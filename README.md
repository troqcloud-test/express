# README

This is the [Express](https://expressjs.com) [Hello world](https://expressjs.com/en/starter/hello-world.html) example on [TroqCloud](https://troqcloud.com).

The app in this repo is deployed at [https://express.troqcloud.app](https://express.troqcloud.app).

## Deployment

See https://troqcloud.com/docs or follow the steps below:

Create a new web service with the following values:
  * Build Command: `yarn`
  * Start Command: `node app.js`

That's it! Your web service will be live on your TroqCloud URL as soon as the build finishes.
